#### Solution:
* Pointers 记录头和尾，通过i迭代
```
class Solution {
    public void sortColors(int[] nums) {
        int l = 0, i = 0, r = nums.length-1;
        
        while (i <= r) {
            if (nums[i] == 2)
                swap(nums, i, r--);
            else if (nums[i] == 0)
                swap(nums, l++, i++);
            else 
                i++;
        }
    }
    
    public void swap(int[] nums, int a, int b) {
        int tmp = nums[a];
        nums[a] = nums[b];
        nums[b] = tmp;
    }
}
```
* Time: O(n)
* Space: O(1)
